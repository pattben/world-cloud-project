# world-cloud-project

This project required [python3](https://www.python.org/download/releases/3.0/) to run.

## Install libraries
```sh
~ pip install tweepy
~ pip install fastapi
~ pip install "uvicorn[standard]"
```
## Run it
```sh
~ uvicorn main:app --reload
```
## Check it

Open your browser at http://127.0.0.1:8000/api/v1/world-clouds
